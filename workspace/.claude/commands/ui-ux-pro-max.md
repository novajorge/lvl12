# UI/UX Pro Max

## Overview
Expert in advanced UI/UX design patterns and best practices for building exceptional user interfaces.

## When to Use
Use this skill for:
- Designing modern, intuitive interfaces
- Improving user experience
- Implementing design systems
- Creating accessible components

## Design Principles

### 1. Visual Hierarchy
- Use size, color, and spacing to guide attention
- Primary actions should be most prominent
- Secondary actions should be subtle

### 2. Consistency
- Use consistent spacing (4px grid system)
- Maintain consistent colors throughout
- Keep typography consistent

### 3. Feedback
- Show loading states
- Provide success/error feedback
- Use appropriate animations

### 4. Accessibility
- Ensure sufficient color contrast (WCAG AA)
- Support keyboard navigation
- Use semantic HTML

## Common Patterns

### Navigation
- Sticky header with navigation
- Breadcrumbs for hierarchy
- Mobile hamburger menu

### Data Display
- Tables with sorting/filtering
- Cards for content items
- Lists with actions

### Forms
- Clear labels and placeholders
- Inline validation
- Helpful error messages

### Loading States
- Skeleton screens
- Progress indicators
- Optimistic updates

## Color Theory
- Primary: Use for main actions
- Secondary: Use for secondary actions
- Success: Green tones for positive feedback
- Error: Red tones for errors
- Warning: Yellow/Orange for cautions

## Typography Scale
- Display: 48px+ (headlines)
- Heading: 24-32px
- Subheading: 18-20px
- Body: 16px
- Small: 14px
- Caption: 12px
